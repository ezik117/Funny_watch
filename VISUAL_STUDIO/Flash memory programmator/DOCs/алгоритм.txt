MCU:
- на старте CS и HOLD должны быть HIGH
- настройка SPI CLK на Rising Edge
- данные вводятся MSB first
- передача инструкций начинается при CS = LOW
- CS дожно быть HIGH после передачи инструкции, иначе такая инструкция может быть проигнорирована
- в процессе выполнения записи в память или ее стирании все инструкции будут игнорироваться кроме инструкции чтения регистра статуса
- MCU должен быть настроен на ручное управление CS


WINDOWS:
- передача по UART идет по два байта:
    - первый байт: AAh-следующий байт - данные для передачи в SPI
                   10-установить CS в 0, 11-установить CS в 1
                   20-установить HOLD в 0, 21-установить HOLD в 1
    - второй байт: данные для записи в SPI
- программа в MCU должна записать в SPI второй байт, если первый байт равен 00
- программа в MCU должна вернуть считанное значение из UART
- программа в MCU должна вернуть считанное значение из SPI
